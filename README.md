# EZRental Auto Rental Management System

## Overview
This project is a multi-tier client-server application designed to manage vehicle rentals, customer data, and transactions.

## Features
- Customer Management (CRUD operations)
- Vehicle Reservation, Rental, and Return System
- Inventory Management
- Credit Card Management System

## Technologies Used
- C# (.NET)
- Microsoft SQL Server
- SQL (DDL, constraints, joins)

## Architecture
- Two-tier desktop application (POS system)
- Three-tier web applications (corporate + customer)
- Shared database layer
